# Logo Inicial - Prospere

Esta pasta contém os logos usados na abertura/apresentação inicial.

## Arquivos

- `prospere-logo.svg` - Logo SVG (preferencial)
- `prospere-logo.png` - Logo PNG (fallback)

## Uso

O logo inicial aparece no **Slide 1** (Abertura) da apresentação.

## Formato

- **SVG**: Escalável, melhor qualidade
- **PNG**: Fallback caso SVG não carregue

## Tamanho Recomendado

- Largura: 300-400px
- Altura: proporcional (geralmente 90-120px)
- Formato: Horizontal/landscape
