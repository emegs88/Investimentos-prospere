# Foto Emerson Gomes dos Santos

Esta pasta deve conter a foto do Emerson da Prospere.

## Arquivo necessário:

- `emerson.jpg` ou `emerson.png` - Foto profissional do Emerson
- Formatos aceitos: JPG, PNG, WebP
- Tamanho recomendado: mínimo 800x800px (quadrado) ou proporção similar
- Fundo: transparente ou neutro (preferencial)

## Uso na apresentação:

A foto será usada automaticamente no **Slide 2** (Quem é Emerson) quando o arquivo for adicionado.

## Nome do arquivo:

Coloque a foto com um dos seguintes nomes:
- `emerson.jpg`
- `emerson.png`
- `emerson.webp`

O componente detecta automaticamente qual formato está disponível.
